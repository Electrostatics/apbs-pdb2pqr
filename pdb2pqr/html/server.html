PDB2PQR Server

Currently using PDB2PQR Version 1.2.0

Mirrors:

  • NBCR at the San Diego Supercomputer Center (UCSD)
  • University College Dublin, Ireland
This server enables a user to convert PDB files into PQR files. PQR files are PDB files where the occupancy and B-factor columns have been replaced by per-atom charge and radius.

pKa calculations are performed by PROPKA.

Note: to distribute server load, the PDB2PQR server currently is limited to a maximum size of 10000 atoms per protein. If you are interested in using PDB2PQR for larger proteins, you are encouraged to download a command line version of PDB2PQR from the PDB2PQR home page.
